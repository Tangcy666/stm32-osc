# STM32F407 数字示波器

基于 STM32F407VETx 微控制器的数字示波器项目，支持高速 ADC 采样、FFT 频谱分析、频率测量和信号发生器功能。

## 项目简介

本项目是一个功能完整的数字示波器系统，通过 STM32F407 的高性能 ADC 和 DAC 外设，实现了信号采集、分析和输出功能。系统支持与上位机或串口屏幕通信，实现波形显示和参数配置。

## 主要功能

### 1. ADC 信号采集

- **多档采样频率**：支持 100Hz、1kHz、10kHz、100kHz、1MHz、7.2MHz 六档采样率
- **DMA 传输**：使用 DMA 实现高效数据传输，减少 CPU 占用
- **三重交替模式**：在 7.2MHz 采样率下，使用 ADC1/ADC2/ADC3 三重交替模式实现高速采样
- **采样深度**：1024 点

### 2. FFT 频谱分析

- **1024 点 FFT**：使用 ARM CMSIS-DSP 库实现快速傅里叶变换
- **频率识别**：自动识别信号主频率
- **频谱显示**：支持将频谱数据发送至上位机显示

### 3. 频率测量

- **输入捕获**：使用 TIM5 定时器输入捕获功能
- **测量参数**：可测量信号频率和占空比
- **高精度**：定时器时钟 2MHz，实现高精度测量

### 4. DAC 信号发生器

- **波形类型**：支持正弦波、方波、三角波、锯齿波
- **频率范围**：1Hz - 500kHz
- **参数可调**：峰峰值、直流偏移、占空比（方波）可调
- **DMA 输出**：使用 DMA 实现波形输出，减少 CPU 占用

### 5. 数据处理

- **IIR 滤波**：对采样数据进行低通滤波，抑制噪声
- **自动测量**：自动计算 Vpp、Vmax、Vmin 等参数
- **自动量程**：根据信号频率自动选择合适的采样率

### 6. HMI 人机交互

- **串口通信**：通过 USART1 与上位机或串口屏幕通信
- **指令解析**：支持多种控制指令，实现参数配置
- **数据上传**：实时上传波形数据和测量结果

## 硬件要求

| 组件   | 规格                 |
| ---- | ------------------ |
| MCU  | STM32F407VETx      |
| 系统时钟 | 144MHz             |
| ADC  | 12-bit, 最高 7.2Msps |
| DAC  | 12-bit, 双通道        |
| 串口   | USART1 (PA9/PA10)  |

## 软件架构

```
STM32F407/
├── Core/                    # STM32CubeMX 生成的核心代码
│   ├── Inc/                 # 头文件
│   └── Src/                 # 源文件
├── Hardware/                # 硬件驱动模块
│   ├── ADC_DMA.c/h          # ADC DMA 采样驱动
│   ├── DAC_DMA.c/h          # DAC DMA 信号发生器驱动
│   ├── HMI.c/h              # 人机交互模块
│   └── message_UART.c/h     # 串口通信模块
├── myCode/                  # 应用层代码
│   ├── Config.c/h           # 配置管理
│   ├── FFT.c/h              # FFT 频谱分析
│   ├── Fre_Measure.c/h      # 频率测量
│   └── OSC_Trigger.c/h      # 触发功能
├── Drivers/                 # STM32 HAL 和 CMSIS 库
├── Middlewares/             # 中间件（ARM DSP 库）
└── MDK-ARM/                 # Keil 工程文件
```

## 串口协议

系统使用 USART1 进行通信，波特率默认 921600。

### 控制指令格式

| 指令   | 功能     |
| ---- | ------ |
| 0xAA | 开启采样   |
| 0xBB | 关闭采样   |
| 0xBC | 自动量程   |
| 0xCC | 开启 FFT |
| 0xDD | 关闭 FFT |
| 0xEE | 开关 DAC |
| 0x01 | 设置采样频率 |
| 0x02 | 设置横轴缩放 |
| 0x10 | 设置波形类型 |
| 0x11 | 设置波形参数 |

## 主要模块说明

### ADC\_DMA 模块

负责 ADC 采样控制，支持单 ADC 模式和三重交替模式。包含数据滤波、缩放等功能。

### DAC\_DMA 模块

实现信号发生器功能，支持多种波形生成，使用 DMA 实现高效输出。

### FFT 模块

使用 ARM CMSIS-DSP 库实现 1024 点实数 FFT，用于频谱分析和频率识别。

### Fre\_Measure 模块

使用定时器输入捕获功能测量外部信号频率和占空比。

### HMI 模块

处理串口通信，解析上位机指令，上传波形和测量数据。
