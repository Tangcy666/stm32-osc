# STM32F407 示波器项目

## 项目概述

这是一个基于STM32F407微控制器的多功能示波器系统，集成了信号采集、波形显示、FFT频谱分析、信号发生器等功能。项目采用模块化设计，具有良好的可扩展性和维护性。

## 主要功能特性

### 信号采集功能
- **高精度ADC采样**：支持最高7.2MHz采样率
- **实时数据处理**：包含滤波、数据缩放、自动测量等功能
- **触发功能**：支持多种触发模式，确保波形稳定显示

### 波形分析功能
- **实时波形显示**：支持750点波形数据实时显示
- **FFT频谱分析**：1024点FFT变换，支持频谱显示
- **自动测量**：自动计算Vpp、频率、占空比等参数
- **横轴缩放**：支持波形横向缩放显示

### 信号发生器功能
- **多种波形输出**：支持正弦波、方波、三角波、锯齿波等
- **可调参数**：频率范围1Hz-100kHz，可调幅值、偏移、占空比
- **DMA输出**：采用DMA方式实现高精度波形生成

### 通信接口
- **UART通信**：与上位机进行数据交互
- **HMI协议**：自定义通信协议，支持参数配置和数据传输

## 硬件平台

### 主控芯片
- **STM32F407**：基于ARM Cortex-M4内核，168MHz主频
- **丰富外设**：内置ADC、DAC、DMA、定时器等外设

### 关键外设配置
- **ADC配置**：
  - 采样率：最高7.2MHz
  - 分辨率：12位
  - 工作模式：多通道交替采样
- **DAC配置**：
  - 输出频率：1Hz-100kHz
  - 波形类型：正弦波、方波、三角波、锯齿波
  - 控制方式：DMA定时器触发
- **定时器配置**：
  - TIM2：DAC波形生成定时器
  - TIM8：ADC采样定时器
  - TIM3/TIM5：频率测量定时器

```
STM32F407/
├── Core/                    # 核心文件
│   ├── Inc/                # 头文件
│   └── Src/                # 源文件
├── Drivers/                 # STM32 HAL驱动库
├── Hardware/               # 硬件驱动层
│   ├── ADC_DMA.[c/h]       # ADC DMA驱动
│   ├── DAC_DMA.[c/h]       # DAC DMA驱动
│   ├── HMI.[c/h]           # 人机交互接口
│   └── message_UART.[c/h]  # 串口通信
├── myCode/                 # 应用层代码
│   ├── Config.[c/h]        # 配置管理
│   ├── FFT.[c/h]           # FFT算法实现
│   ├── Fre_Measure.[c/h]   # 频率测量
│   └── OSC_Trigger.[c/h]   # 触发控制
└── MDK-ARM/               # Keil工程文件
```

### 核心模块说明

#### 1. 主程序模块 (main.c)
- 系统初始化和外设配置
- 主循环调度和任务管理
- 数据采集和处理流程控制

#### 2. ADC DMA驱动 (ADC_DMA.[c/h])
- ADC采样配置和启动
- DMA数据传输管理
- 数据滤波和预处理
- 多通道数据转换

#### 3. DAC DMA驱动 (DAC_DMA.[c/h])
- DAC波形生成配置
- 多种波形算法实现
- DMA输出控制
- 参数动态调整

#### 4. FFT分析模块 (FFT.[c/h])
- 快速傅里叶变换实现
- 频谱数据计算
- 频率分量分析

#### 5. 频率测量模块 (Fre_Measure.[c/h])
- 基于定时器的频率测量
- 占空比计算
- 测量精度校准

#### 6. 触发控制模块 (OSC_Trigger.[c/h])
- 触发条件设置
- 波形稳定控制
- 触发模式管理

#### 7. HMI通信模块 (HMI.[c/h])
- 上位机通信协议
- 数据帧格式定义
- 命令解析和处理

- **输出频率**：1Hz-100kHz

## 使用说明

### 基本操作流程
1. **系统启动**：上电后系统自动初始化
2. **参数配置**：通过串口发送配置命令
3. **数据采集**：启动ADC采集波形数据
4. **数据分析**：实时显示波形和测量参数
5. **信号输出**：配置DAC输出测试信号

### 通信协议
项目使用自定义的HMI通信协议，主要命令包括：
- 系统启停控制
- 采样率设置
- 触发参数配置
- 数据请求和传输
- DAC参数设置

4. **通信速率**：确保串口通信速率匹配
