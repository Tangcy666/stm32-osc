# STM32F407 数字示波器项目

## 项目概述

这是一个基于STM32F407微控制器的数字示波器项目，具备波形采集、FFT频谱分析、DAC信号输出等功能。项目使用STM32CubeMX生成基础代码框架，并在此基础上实现了完整的示波器功能。

## 主要功能特性

### 1. 波形采集功能
- **多通道ADC采样**：支持ADC1、ADC2、ADC3三通道采样
- **高采样率**：最高支持7.2MHz采样频率
- **DMA传输**：使用DMA实现高效的数据传输
- **多模式采样**：支持单通道和多模式采样配置

### 2. 信号处理功能
- **数字滤波**：实现IIR滤波和快速移动平均滤波
- **自动测量**：自动测量信号的峰峰值、频率、占空比等参数
- **横轴缩放**：支持波形横轴的缩放显示

### 3. FFT频谱分析
- **实时FFT**：基于ARM CMSIS-DSP库实现快速傅里叶变换
- **频谱显示**：提供频谱图的实时显示功能
- **频率测量**：通过FFT精确测量信号频率

### 4. DAC信号输出
- **波形生成**：支持多种波形输出（正弦波、方波、三角波等）
- **参数可调**：可调节输出波形的幅度、频率、偏移量等参数
- **DMA控制**：使用DMA实现平滑的波形输出

### 5. 人机交互界面
- **串口通信**：通过UART与上位机进行数据通信
- **命令解析**：支持多种控制命令的解析和执行
- **数据帧传输**：实现波形数据和测量结果的实时传输

## 硬件配置

### 微控制器
- **型号**：STM32F407VETx
- **主频**：168MHz
- **内存**：192KB RAM, 512KB Flash

### 外设配置
- **ADC**：ADC1、ADC2、ADC3三通道，最高7.2MHz采样率
- **DAC**：DAC通道1，用于信号输出
- **定时器**：TIM2、TIM3、TIM5、TIM8用于各种定时和PWM功能
- **UART**：USART1用于与上位机通信
- **DMA**：多通道DMA用于ADC和DAC数据传输

## 软件架构

### 主要模块

1. **ADC_DMA模块**
   - 负责ADC的初始化和DMA配置
   - 实现多模式采样和数据转换
   - 提供数据滤波和缩放功能

2. **DAC_DMA模块**
   - 管理DAC输出和波形生成
   - 支持多种波形类型和参数调节

3. **FFT模块**
   - 基于ARM CMSIS-DSP库实现FFT运算
   - 提供频谱分析和频率测量功能

4. **HMI模块**
   - 处理与上位机的通信协议
   - 解析控制命令和数据传输

5. **Config模块**
   - 管理系统配置参数
   - 管理测量数据存储

## 使用说明

### 基本操作流程

1. **系统启动**：上电后系统自动初始化所有外设
2. **连接上位机**：通过串口连接上位机软件
3. **配置参数**：发送配置命令设置采样率、触发模式等
4. **开始采样**：启用采样功能开始波形采集
5. **数据分析**：查看实时波形或进行FFT分析
6. **信号输出**：配置DAC参数输出测试信号

### 通信协议

项目使用自定义的串口通信协议，主要命令包括：
- 系统配置命令
- 采样控制命令
- 数据请求命令
- DAC输出控制命令

## 故障排除

### 常见问题

1. **采样数据异常**
   - 检查ADC参考电压
   - 验证信号调理电路
   - 确认采样率设置

2. **通信连接失败**
   - 检查串口波特率设置
   - 验证硬件连接
   - 确认通信协议格式

3. **FFT结果不准确**
   - 检查采样率与信号频率的匹配
   - 验证窗函数选择
   - 确认FFT点数设置
